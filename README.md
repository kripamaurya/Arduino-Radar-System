# 📡 Arduino Radar System

This is a DIY Arduino Radar System using an Ultrasonic Sensor and Servo Motor to detect objects and measure distance.

---

## 🚀 Features
- Detects objects using ultrasonic sensor  
- Measures distance in real-time  
- Servo motor rotates from 0° to 180°  
- Works like a mini radar system  

---

## 🧰 Components Used
- Arduino Uno  
- HC-SR04 Ultrasonic Sensor  
- SG90 Servo Motor  
- Jumper Wires  

---

## ⚙️ Working
The servo motor rotates the ultrasonic sensor from 0° to 180°.  
The sensor measures distance and sends data to Arduino.  
Arduino processes and displays the data.
